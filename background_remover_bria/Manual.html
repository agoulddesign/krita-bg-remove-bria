Background Remover BriaAI v1.2

A simple plugin for Krita that quickly and automatically removes the background of images using BriaAI's API. No need to load a third party app or website every time you need a subject isolated.

Usage:

  1. You must first create an account and an API key on https://bria.ai/. Copy and paste the API key in the API key field.
  2. Select the layer(s) you want to remove the background of and click Remove Background (it may take a few seconds). It will then create a new layer with an alpha channel and background removed.
  3. The removal process doesn't work very well on images with depth of field or with a lot of subjects/objects in them, or if they're on many different levels (i.e., foreground, midground, background). If the removal doesn't work quite as expected, you can transfer the alpha of the new layer with the background removed to the original layer as a transparency mask and adjust accordingly.

I hope you enjoy the power and ease of automatic background removal in Krita!

Please note this only works in RGBA color space

Visit https://github.com/agoulddesign/krita-bg-remove-bria for the most recent version.
